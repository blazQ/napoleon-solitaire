# Napoleon Solitaire
In this repository you'll find code for an extremely basic implementation of a Solitaire game in Godot.
It defines scenes and scripts for moving cards as 3d objects in 3d space, assigning them to different piles based on the rule of the implemented game (in this case, the variant is called Napoleon Solitaire) and contains very basic UI.
It's mainly developed as a personal introduction to the Godot Engine and its features.
